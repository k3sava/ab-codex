# Full-Funnel Domain Map (AI + Non-AI)

## Goal
Cover every major growth domain needed to attract, educate, acquire, adopt, retain, and grow customers.

## Domain set
- Content Marketing
- SEO / AEO
- Social Media Marketing
- PR / Communications
- Brand
- Design
- Event Marketing
- Inbound
- Outbound
- Unbound Marketing
