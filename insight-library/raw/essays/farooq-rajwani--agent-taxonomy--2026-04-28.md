# Excerpt (from miniu digest 2026-04-29)

Hamza Farooq and Jaya Rajwani (guest piece in Lenny's Newsletter) taxonomize agent backlogs into three structurally different systems:

1. **Deterministic automation** (n8n, Zapier) — fixed graph, no reasoning step.
2. **Reasoning-and-acting agents** (LangGraph, CrewAI) — single agent with a tool loop.
3. **Multi-agent networks** — multiple agents coordinating, with emergent behavior.

> "One agent might take six weeks to build. Another might take six months."

The cost asymmetry is real. Tagging each agent loop in a roadmap with its class lets resourcing match the cost band rather than treating "agent" as one estimate.
